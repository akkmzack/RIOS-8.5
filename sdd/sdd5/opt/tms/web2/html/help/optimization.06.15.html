Configuring CIFS Optimization
This section describes how to optimize CIFS. It includes these sections:
  • Optimizing CIFS SMB1
  • Optimizing SMB2/3
  • Configuring SMB Signing
    • You display and modify CIFS optimization and SMB Signing settings in the Configure > Optimization > CIFS (SMB1) page and the Configure > Optimization > SMB2/3 pages.
    CIFS Enhancements by Version
    This section lists and describes new CIFS features and enhancements by RiOS version.
  • RiOS v8.5 provides support for SMB3 latency and bandwidth optimization.
  • RiOS v8.0 provides support for SMB1 signing settings for Mac OS X Lion (10.7) and Mountain Lion (10.8). RiOS v8.0 does not support SMB2 signing settings for Mac OS X Lion (10.7) and Mountain Lion (10.8).
  • RiOS v7.0 and later simplifies configuration for SMB1 and SMB2 latency optimization in Windows 7 when you configure the server-side Steelhead appliance in Active Directory integrated mode for Windows 2003 or Windows 2008. After joining in AD integrated mode, you can use transparent mode instead of delegation mode.
  • RiOS v6.5 supports SMB2 latency optimization.
    • CIFS latency optimization does not require a separate license and is enabled by default.
    Typically, you disable CIFS optimizations only to troubleshoot the system.
