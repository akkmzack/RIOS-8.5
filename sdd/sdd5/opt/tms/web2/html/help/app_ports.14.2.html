Appendix B Steelhead Appliance Ports : Granite Ports
Granite Ports
This table lists and describes the Steelhead EX Granite default ports with the port label Granite.
Default Ports
Description
7950
Data requests for data blocks absent in Edge appliance from the data center
7951
New data created at the Edge to the data center
7952
Prefetch data for which Granite has highest confidence (for example, file read ahead)
7953
Prefetch data for which Granite has medium confidence (for example, boot)
7954
Prefetch data for which Granite has lowest confidence (for example, prepopulation)
7970
Management information exchange between Edge and Core appliances
