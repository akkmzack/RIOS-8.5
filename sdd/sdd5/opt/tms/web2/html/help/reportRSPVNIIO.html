Chapter 10 Viewing Reports and Logs : Viewing RSP Statistics Reports
Viewing RSP Statistics Reports
The RSP VNI IO report shows the throughput (in various network directions) of a selected VNI for the time period specified.
For details on the report format, see Overview.
RSP statistics reports appear only on a Steelhead appliance xx20 or xx50.The RSP Statistics report contains these statistics that summarize RSP activity.
Data Series
Description
LAN In
Indicates the rate of bps or packets per second coming into the LAN.
LAN Out
Indicates the rate of bps or packets per second going out of the LAN.
WAN In
Indicates the rate of bps or packets per second coming into the WAN.
WAN Out
Indicates the rate of bps or packets per second going out of the WAN.
Package In
Indicates the rate of bps or packets per second coming into the RSP package.
Package Out
Indicates the rate of bps or packets per second going out of the RSP package.
The navigator shadows the LAN In series.
What This Report Tells You
The RSP VNI IO report answers these questions:
  • How much traffic is coming in and going out the LAN direction?
  • How much traffic is coming in and going out the WAN direction?
  • How much traffic is coming in and going out of a package?
    • About Report Graphs
    Use the mouse to hover over a specific data point to see what the y values and exact timestamp were in relation to peaks.
    About Report Data
    The Riverbed system reports on performance for periods up to one month. Due to performance and disk space considerations, the display granularity decreases with time passed since the data was sampled. The data is collected at a 5-minute granularity for the entire month.
    To view the RSP VNI IO report
    1.
    Choose Reports > Branch Services > RSP VNI IO to display the RSP VNI IO page.
    2.
    Use the controls to customize the report as described in this table.
    Control
    Description
    Time Interval
    Select a report time interval of 5 minutes (5m), 1 hour (1h), 1 day (1d), 1 week (1w), All, or type a custom date. All includes statistics for the last 30 days.
    Time intervals that do not apply to a particular report are dimmed.
    For a custom time interval, enter the start time and end time using the format YYYY/MM/DD HH:MM:SS.
    You can quickly see the newest data and see data points as they are added to the chart dynamically. To display the newest data, click Show newest data.
    VNI
    Select a VNI from the drop-down list.
    Units
    Select either packets/sec or bps from the drop-down list.
