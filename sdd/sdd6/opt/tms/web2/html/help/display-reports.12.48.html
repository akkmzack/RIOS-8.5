Chapter 10 Viewing Reports and Logs : Downloading Log Files : Downloading System Log Files
Downloading System Log Files
You can download system logs in the System Logs Download page. Download system logs to monitor system activity and to troubleshoot problems.
To download system logs
1.
Choose Reports > Diagnostics > System Logs Download to display the System Logs Download page.
2.
Select the name of the log to display the dialog box to display or save the log to disk.
3.
Click Rotate Logs to archive the current log to a numbered archived log file and then clear the log so that it is empty again.
