Chapter 10 Viewing Reports and Logs : Exporting Performance Statistics
Exporting Performance Statistics
You can export performance statistics in CSV format in the Export report. The CSV format allows you to easily import the statistics into spreadsheets and databases. You can open the CSV file in any text editor.
The CSV file contains commented lines (comments beginning with the # character) at the beginning of the file. These comments report what host generated the file, the report that was generated, time boundaries, the time the export occurred, and the version of the Steelhead appliance the file was exported from. The statistical values are provided in columns: the first column is the date and time of the statistic sample, the columns that follow contain the data.
To export statistics
1.
Choose Reports > Export to display the Export page.
2.
Use the controls to customize the report, as described in the following table.
Control
Description
Report
Select the type of report you want to export from the drop-down list.
Begin Date and Time
Specify a start date and time in the following format:
YYYY/MM/DD HH:MM:SS
The report can contain up to 2 days of traffic statistics, depending on how long the service has been up and the traffic volume.
End Date and Time
Specify an end date and time in the following format:
YYYY/MM/DD HH:MM:SS
The report can contain up to 2 days of traffic statistics, depending on how long the service has been up and the traffic volume.
Email Delivery
Sends the report to an email address.
Email Address
Specify the email address of the recipient.
Export
Exports the report data.
